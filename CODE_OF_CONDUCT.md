# Code of Conduct

## Our Standard

这个项目希望保持直接、专业、尊重事实的协作方式。

请遵守这些基本原则：

- 尊重他人，不做人身攻击
- 讨论代码、行为和结果，不揣测动机
- 允许提出强烈异议，但请给出可验证依据
- 不发布骚扰、歧视、威胁或羞辱性内容
- 不故意刷屏、引战或破坏讨论秩序

## Scope

这份行为规范适用于：

- issue
- pull request
- discussion
- review comment
- 与本仓库直接相关的公开协作场景

## Enforcement

维护者可以根据情况采取这些措施：

- 提醒或要求修改表达方式
- 隐藏或删除不当内容
- 暂时限制互动
- 在严重情况下拒绝继续协作

## Reporting

如果你遇到明显违反这份规范的行为，请优先通过维护者提供的私密联系渠道反馈；如果没有私密渠道，再根据场景决定是否公开说明。
